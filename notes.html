JavaScript Notes

JavaScript (JS) is a programming language that allows developers to create interactive web pages. It's used to make web pages more engaging by adding features like animations, interactive maps, and dropdown menus.

What JavaScript does

  • Makes static content dynamic
  • Displays timely content updates
  • Creates interactive maps
  • Animates 2D/3D graphics
  • Creates scrolling video jukeboxes
  • Creates pop-ups
  • Creates image carousels
  • Creates click-to-show dropdown menus
  • Creates dynamically changing element colors

Variables

Variables are the name of memory location where we can store the data.

Example:

let name = "Aryan";
const age = 19;

Data Types

JavaScript supports various data types including:

  • String
  • Number
  • Boolean
  • Object
  • Undefined
  • Null
  • Symbol (ES6)

Example:

let str = "Hello, Aryan jha!"; // String
let num = 100; // Number
let bool = true; // Boolean
let obj = { name: "Aryan jha", age: 19, Roll no: 04}; // Object
let undef; // Undefined
let nul = null; // Null
let sym = Symbol("id"); // Symbol

Comments

Comments are used to explain code and are ignored by the JavaScript engine. There are single-line and multi-line comments.

Example:

// This is a single-line comment

/*
This is a
multi-line comment
*/

Identifier Rules

Identifiers are names given to variables, functions, etc. The rules for identifiers are:

  • Must start with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can be letters, digits, underscores, or dollar signs.
  • Cannot be a reserved keyword.

Example:

let _name = "Aryan jha";
let $age = 19;
let user1 = "Raj";

Operators and Precedence

Operators are used to perform operations on variables and values. Operator precedence determines the order in which operators are evaluated.

Example:

let x = 5;
let y = 10;
let z = x + y * 2; // Multiplication (*) has higher precedence than addition (+)

Operator Precedence Table:

  1. Parentheses ()
  2. Increment/Decrement ++ --
  3. Multiplication/Division/Modulus * / %
  4. Addition/Subtraction + -
  5. Relational < <= > >=
  6. Equality == != === !==
  7. Logical AND &&
  8. Logical OR ||
  9. Assignment = += -= *= /=
